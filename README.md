# Tetris_game
A portable tetris game to play the iconic game in the terminal or in a hardware interface

>### Install game 
> * open a terminal, and move to the directory when you gonna save the game
>   
>       cd ~/tetris_game
>
>* clone the repository in the directory
>
>        git clone https://github.com/DanielRobayo12/tetris_game.git
>
>* open proyect in platformio
>
>Open visual studio -> install platformio extension -> pick open project in PIO Home -> select the folder "tetris_game"
